## Development

### Setup

Run this on first time

- Clone Repository

```shell
git clone https://github.com/NCHU-NLP-Lab/API.git
cd API
```

- Install Dependencies

```shell
pip install -r requirements.txt
# or use poetry
poetry install
```

### Launch

```shell
uvicorn server:app --reload
```
